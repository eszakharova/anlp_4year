## Статистики
1. LogLikeliHood

2. Weirdness - частотная доля
   употребления выражения в коллекции по сравнению с употреблением в контрастной коллекции

   ```
   (abs_freq_corp(word)/len_corp)/(abs_freq_ref(word)/len_ref)
   ```
